# ASSA

**ASSA** — 構音障害者向け日本語音声コーパス（JDSC）プロジェクト

構音障害当事者であった小笠原朝陽の遺志を継ぎ、「どんな声も、置き去りにしない社会」の実現を目指す研究プロジェクト。

## Structure

```
assa/
├── web/       ← プロジェクト LP（GitHub Pages）
├── papers/    ← 論文 PDF
├── data/      ← コーパスデータ等
├── docs/      ← 議事録・メモ
└── assets/    ← 画像素材
```
